# LUMORA MVP v0.1

## Objective

Develop a professional SaaS platform that demonstrates Full Stack development skills using Java, Spring Boot and Angular, focused on transforming business data into actionable insights.

## Target User

Business managers and executives who need to monitor business performance and make data-driven decisions.

## Functional Modules

- Authentication
- Companies
- Stores
- Products
- Inventory
- Sales
- Executive Dashboard
- KPI Management

## Core Features

- User login
- Dashboard with KPIs
- Interactive charts
- Inventory management
- Sales management
- CSV data import
- Responsive interface
- REST API documentation with OpenAPI / Swagger

## Technical Stack

- Java 21
- Spring Boot
- Angular
- PostgreSQL
- Spring Security
- JPA / Hibernate
- Docker
- OpenAPI / Swagger
- JUnit
- Mockito
- Maven
- Git
